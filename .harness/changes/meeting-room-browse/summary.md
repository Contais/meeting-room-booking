# 会议室浏览 - 需求分析（回顾性）

## 需求描述
普通用户浏览可用会议室列表，查看会议室详情，了解设备和容纳人数。

## 验收标准
| 编号 | 验收条件 |
|------|----------|
| AC-1 | 用户可查看所有启用的会议室列表（卡片布局） |
| AC-2 | 支持按名称/位置搜索、按最少容纳人数筛选 |
| AC-3 | 点击会议室可查看详情（名称、位置、容量、设备、描述） |
| AC-4 | 详情页显示使用规则信息 |

## 技术变更
| 变更项 | 文件 | 说明 |
|--------|------|------|
| MeetingRoomController | GET /meeting/room/list | 公开接口，仅返回启用状态的会议室 |
| MeetingRoomController | GET /meeting/room/{id} | 公开接口，返回会议室详情 |
| RoomListView.vue | 前端页面 | 卡片网格布局 + 筛选栏 |
| RoomDetailView.vue | 前端页面 | 左右两栏布局 + 规则信息 + 预约入口 |

## 涉及模块
- mrb-meeting（后端公开接口）
- frontend（浏览页面）
