# 接口协议 (API Protocol)

## 1. 通用响应格式

所有API响应必须遵循以下结构：


### 1.1 响应码定义

| 响应码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 参数错误 |
| 401 | 未授权 |
| 403 | 禁止访问 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |
| 1001 | 业务错误：库存不足 |
| 1002 | 业务错误：商品已下架 |
| 1003 | 业务错误：优惠券无效 |

## 2. 核心接口清单

### 2.1 商品接口

#### GET /api/product/{productId}
获取商品详情

**响应参数：**
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Long | 商品ID |
| name | String | 商品名称 |
| categoryId | Long | 分类ID |
| mainImage | String | 主图URL |
| priceCents | Integer | 现价（分） |
| originalPriceCents | Integer | 原价（分） |
| stock | Integer | 库存 |
| skus | Array | SKU列表 |

#### GET /api/product/search
商品搜索

**请求参数：**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| keyword | String | 否 | 关键词 |
| categoryId | Long | 否 | 分类ID |
| page | Integer | 否 | 页码 |
| size | Integer | 否 | 每页大小 |

### 2.2 购物车接口

#### POST /api/cart/add
添加商品到购物车

**请求参数：**
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| skuId | Long | 是 | SKU ID |
| quantity | Integer | 是 | 数量 |
| userId | Long | 否 | 用户ID（登录用户必填） |

#### GET /api/cart
获取购物车列表

**响应参数：**
| 字段 | 类型 | 说明 |
|------|------|------|
| items | Array | 购物车项列表 |
| totalAmountCents | Integer | 总金额（分） |

### 2.3 订单接口

#### POST /api/order/create
创建订单

**请求参数：**
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| cartItemIds | Array | 是 | 购物车项ID列表 |
| addressId | Long | 是 | 收货地址ID |
| couponId | Long | 否 | 优惠券ID |

**响应参数：**
| 字段 | 类型 | 说明 |
|------|------|------|
| orderId | Long | 订单ID |
| totalAmountCents | Integer | 订单总金额（分） |
| payAmountCents | Integer | 实付金额（分） |
