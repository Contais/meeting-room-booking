# 工程结构规范

## 前后端分离项目目录结构

### Vue 前端（`frontend/`）

src/
├── assets/            # 静态资源
├── components/        # 公共组件
├── composables/       # 组合式函数
├── layouts/           # 布局组件
├── router/            # 路由配置
├── stores/            # Pinia 状态管理
├── styles/            # 全局样式
├── types/             # TypeScript 类型定义
├── utils/             # 工具函数
└── views/             # 页面视图


### Spring Boot 后端 (`backend/`)

src/main/java/com/meetinghub/
├── config/            # 配置类
├── controller/        # 控制器层
├── service/           # 服务层
│   └── impl/          # 服务实现
├── repository/        # 数据访问层
├── model/             # 模型层
│   ├── entity/        # 数据库实体
│   ├── dto/           # 数据传输对象
│   └── vo/            # 视图对象
├── mq/                # 消息队列
│   ├── producer/      # 消息生产者
│   └── consumer/      # 消息消费者
└── common/            # 通用模块
    ├── exception/     # 异常定义
    ├── enums/         # 枚举类
    ├── constant/      # 常量定义
    └── util/          # 工具类


## 四层架构

Controller -> Service -> Repository -> Model

| Layer      | Responsibility          | Key Annotation        |
| :--------- | :---------------------- | :-------------------- |
| Controller | 请求路由、参数校验、响应构造 | @RestController     |
| Service    | 业务逻辑、事务管理       | @Service            |
| Repository | 数据访问、SQL 映射       | @Mapper / BaseMapper |
| Model      | 数据结构定义             | @TableName / POJO    |

**依赖方向**: Controller -> Service -> Repository -> Model, 严禁反向依赖。
